# How to set your birthday

1. Go to "#--birthdays--" slack channel
2. Run "/birthday_register" to register your birthday
3. It will automatically wish you on 12:00 PM on your timezone (or the one you set in your slack acc)

# How to delete your birthday

1. Go to "#--birthdays--" channel
2. Run "/birthday_delete"
3. Confirm again in the bot message
4. It is deleted
